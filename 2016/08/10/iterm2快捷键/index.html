iterm2快捷键

  • MacOS
  • OSX
  • iterm2
快捷键 功能
⌘ + t 新建标签
⌘ + w 关闭标签
⌘ + 1-9 切换标签
⌘ + →,← 切换标签
⌘ + w 关闭标签
⌘ + d 垂直分屏
⌘ + ⇧ + d 水平分屏
⌘ + ; 查看历史命令
⌘ + ⇧ + h 查看剪贴板历史
⌘ + f 查找
⌃ + u 清除当前行
⌃ + a 到行首
⌃ + e 到行尾
⌃ + f 前进
⌃ + b 后退
⌃ + p 上一条命令
⌃ + r 搜索命令历史
⌃ + d 删除当前光标的字符
⌃ + h 删除光标之前的字符
⌃ + w 删除光标之前的单词
⌃ + k 删除到文本末尾
⌃ + t 交换光标处文本
⌘ + r 清屏1
⌃ + l 清屏2

⌘(command)
⌃(ctrl)
⇧(shift)
